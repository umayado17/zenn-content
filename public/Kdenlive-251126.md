Kdenliveは、KDE e.V.（ドイツ非営利団体）が主導する無料・オープンソースの動画編集ソフトウェアです。プロ用途にも耐える豊富な機能を持ちながら、透明性の高い運営とローカル完結の設計により、プライバシー・セキュリティ面で高い信頼性を示しています。

- 対象ソフトウェア: Kdenlive
- 公式URL: [https://kdenlive.org/](https://kdenlive.org/)
- 安全性レベル: 4（安全）
- 厚黒学レベル: 0/18項目
- 支配国名目国: ドイツ

## エグゼクティブ・サマリー

Kdenliveは、長年の実績を持つKDEコミュニティによって開発される動画編集ソフトウェアで、透明性・信頼性の観点から高く評価できます。

- **法務判定**: 導入可（適切な設定管理下で推奨）
- **技術判定**: 安全
- **主要な強み**:
  - ドイツ非営利団体主導のオープンソースプロジェクト
  - ローカルPC完結の設計、クラウド送信なし
  - GPL-3.0ライセンス、ソースコード完全公開
  - テレメトリはオプトイン方式（明示的許可が必要）
  - AI機能もローカル実行が基本
- **軽微な注意点**:
  - 音声認識エンジンVOSKがロシア企業製（ただしOSS、ローカル実行、任意使用）

## 詳細調査結果

### 技術アーキテクチャ分析

#### コア技術スタック

**開発言語・フレームワーク**[^1]:
- C++ / Qt / KDE Frameworks（KF5）ベースのデスクトップアプリケーション
- クロスプラットフォーム対応（Linux, Windows, macOS, BSD）

**マルチメディア処理**[^2]:
- MLT Framework（Media Lovin' Toolkit）をベースエンジンとして採用
- FFmpeg、Frei0r、SoX、LADSPAなどのオープンソースライブラリを活用
- 多様なフォーマット対応（AVI, MPEG, MP4, WMV等、SD/HD/4K）

**配布形態**[^3]:
- 各Linuxディストリビューションのリポジトリ
- Flatpak/Snap（サンドボックス化されたパッケージ）
- Windows用インストーラー、macOS用パッケージ
- すべてGPL系ライセンスで再配布自由

#### AI関連機能の実装

Kdenliveは近年、AI技術を限定的に統合していますが、**ローカル実行を基本**とした設計です[^4]。

**音声認識・自動字幕生成**:
- **VOSK**（Alpha Cephei社製、Apache 2.0ライセンス）
  - オフライン音声認識ツールキット
  - モデルファイルをローカルにダウンロードして使用
  - インターネット接続不要で動作
- **Whisper**（OpenAI製）
  - ローカルインストール版のPythonモジュールを使用
  - デフォルトでクラウドAPI接続なし

**翻訳機能**:
- **SeamlessM4T**（Meta製）
  - 音声→音声/テキスト翻訳
  - 実装によってはオンラインサービス接続の可能性あり
  - ユーザー設定次第で外部API利用が発生しうる

**重要な設計思想**:
- AI機能は**すべてオプション**（使用しなくても全機能利用可能）
- デフォルトはローカル実行
- 外部APIへの接続は明示的な設定が必要

### 法的条項分析

#### ライセンス

**GPL-3.0-or-later**[^5]:
- フリーソフトウェアライセンス
- ソースコード公開義務
- 商用利用可能
- 再配布・改変自由（同ライセンス継承が条件）

**組織利用時の注意**:
- カスタム版を配布する場合、GPL条件の理解が必要
- 単純な社内・個人利用では制約は緩い
- 商用プロジェクトでの利用も可能

#### プライバシーポリシー

**KDE e.V.のテレメトリポリシー**[^6][^7]:

```markdown
基本原則:
- オプトイン方式（ユーザーの明示的許可が必要）
- ユニークIDは使用しない
- コンテンツデータは収集しない
- 第三者への提供は行わない
```

**収集される情報（許可した場合のみ）**:
- 匿名化された使用統計
- システム情報（OS、CPU、メモリ等）
- クラッシュレポート（オプション）

**KUserFeedbackフレームワーク**[^8]:
- KDEアプリケーション用の統一テレメトリシステム
- ユーザーによる完全な無効化が可能
- 収集データの透明性確保

**データ保存場所**:
- 動画・プロジェクトファイル: ユーザーのローカルストレージのみ
- クラウド保存機能なし（標準機能として）
- 外部サーバへのアップロード機能なし

**適用法令**:

## 推奨対応

### ✅ 推奨される使用方法

#### 個人ユーザー向け

**標準的な使用（最も安全）**:
```
✓ デフォルト設定での使用
✓ AI機能はローカル実行（VOSK/Whisperローカル版）
✓ テレメトリは無効化 or 最小化
✓ 外部API連携は避ける
✓ 公式リポジトリからのダウンロード
```

**AI機能使用時の推奨設定**:
```
音声認識:
- 第1推奨: Whisperローカル版
- 第2推奨: VOSKローカル版
- 非推奨: 外部APIの使用

翻訳機能:
- ローカル実行可能なモデルのみ使用
- 外部API接続は避ける
```

#### 企業・組織での導入時

**セキュリティポリシーの設定**:
```yaml
必須設定:
  - AI機能の外部API使用禁止
  - テレメトリ完全無効化
  - ローカルモデルのみ使用許可
  - インターネット接続の制限（可能な場合）

推奨設定:
  - Flatpak/Snap版での配布（サンドボックス化）
  - 定期的なアップデート管理
  - 使用ログの監査
```

**導入時チェックリスト**:
- [ ] セキュリティポリシーの策定
- [ ] AI機能使用ガイドラインの作成
- [ ] テレメトリ無効化手順の文書化
- [ ] 代替ソフトウェアの評価（比較検討）
- [ ] 定期的な見直しスケジュールの設定

## 最終総括

Kdenliveは、**西側主導のオープンソースプロジェクトとして高い信頼性**を持つ動画編集ソフトウェアです。

### 主要な評価ポイント

**✅ 強み**:
1. **透明性**: ドイツ非営利団体主導、20年の実績、完全なソースコード公開
2. **プライバシー**: ローカル完結設計、クラウド送信なし、テレメトリはオプトイン
3. **地政学的安全性**: EU法制下、中国との関係なし、民主的ガバナンス
4. **技術的信頼性**: 成熟したコードベース、活発なコミュニティ、定期的更新
5. **コスト**: 完全無料、GPL、商用利用可能

**⚠️ 注意点**:
1. **VOSK問題**: ロシア企業製（ただしOSS、ローカル実行、任意使用）
2. **AI外部API**: 設定次第で外部サービス接続の可能性
3. **OSS特有のリスク**: サプライチェーン攻撃の理論的可能性

### リスク・ベネフィット分析

**リスク**:
- VOSKのロシア起源（実質的リスクは限定的）
- AI機能の設定ミスによる外部API接続
- OSS特有のサポート体制

**ベネフィット**:
- 完全無料で高機能
- プライバシー保護に優れた設計
- 透明性の高い運営
- 地政学的リスクが低い

### 最終判定

**安全性レベル: 4（安全）**

Kdenliveは、適切な設定管理（AI機能のローカル実行、テレメトリ無効化等）を行えば、**企業・個人を問わず推奨できる動画編集ソフトウェア**です。

VOSKのロシア起源という点は認識すべきですが:
- オープンソースでコード検証可能
- ローカル実行（データ送信なし）
- 任意使用（Whisper等の代替あり）

という特性から、**実質的なリスクは限定的**と評価できます。

**導入判定**: ✅ **推奨**（適切な設定管理下で）

特に、Adobe等のクラウド連携強制型ソフトウェアからの移行先として、**プライバシーとセキュリティを重視する組織に適している**と言えます。
